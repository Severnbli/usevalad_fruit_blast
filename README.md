## Description

2D arcade made with Unity where fruits explode, bounce, and shower XP.

### Features

* Lifecycle state machine.
* Custom physics engine.
* Gyroscope support.

### Used utils

* [UniTask](https://github.com/Cysharp/UniTask)
* [DOTween](https://dotween.demigiant.com/)
* [Odin](https://odininspector.com/)

## Getting Started

### Building

1. Download project via `git clone` (preferred) or [as .zip file](https://github.com/Severnbli/usevalad_fruit_blast/archive/refs/heads/main.zip).
2. Go to Unity and open project. Project unity version is `2022.3.45f1`.
3. Download and install [Odin](https://odininspector.com/) package.
4. In Editor go to File → Build Profiles.
5. Build project.
